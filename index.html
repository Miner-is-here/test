Prace s GITem

  • U = untracked
  • Polozky cekaji, az se vlozi do historie - commit znamena vlozit do historie - je vhodne si vlozit komentar.
  • M = modified
  • A = added, tzn. pokud commitnu soubor, tak se z U zmeni na A, pokud modifikuju, tak z A na M
  • .gitignore viz hesla.txt
  • github = prace od microsoftu, ktera dovoluje nahravat veci online
  • po nahrani muzu bud poslat jenom link nebo link pres code - nekomu ale kdo umi programovat a ma github
  • pokud bych chtel poslat stranku klientovi, tak ten kodu nerozumi, muze se podvat pres demo rezim - k tomu slouzi git pages, musim kliknout na setting, dole vlevo pages a tam musim rict, kde je soubor index.html
